ARENA / IDENTITY SYSTEM

Ring prototypes
10 EXPERIMENTS
GITHUB · the-arena
GIT TRACKED
All prototypes share the canonical 4-ring mark geometry from RINGS.svg. Each pushes a different axis of the system — manipulation, motion, decomposition, multiplicity, deformation. No surface treatments. Pure geometry.
01 / PLAYGROUND
Ring Explorer
Manipulable canonical mark with 12 form presets.
Sliders for ring count, envelope dimensions, band thickness (separate V/H), inter-ring gap, corner radius. Export SVG. The system-defining tool.
SLIDERS · R RESET · G GRID · I INVERT
02 / MOTION
Single Mark Morphing
One mark drifting between 14 moods.
Director cycles through Canonical, Wide Open, Pillar, Many, Pill, Square, Atomic, Alert, Sliver, Overscale, Offset, Inversion, Forge, Drift. Per-ring breathing on top.
F FULL · SPACE PAUSE · N NEXT · H HUD
03 / INDEPENDENT
Decoupled Ring Motion
4 rings transform on their own, then re-lock.
Each ring rendered as its own group with independent transform. 14 compositions — drift, scatter, orbit, twist, asymmetric — periodically returning to canonical as the brand "lock" moment.
F FULL · SPACE PAUSE · N NEXT · C LOCK
04 / FORGE NEW
Cursor Forge
Rings deform around the cursor via spring physics.
Each ring has its own mass/stiffness. The cursor exerts inverse-square force. Four modes: PUSH, PULL, ORBIT, DISTORT. Hold to forge. Release: springs back to canonical.
MOVE STEER · HOLD FORGE · M MODE · H HUD
05 / STRESS NEW
Stress System
Pressure deforms the mark. Release snaps back hard.
Six deformation modes — CRUSH, STRETCH, SHEAR, BUCKLE, TWIST, RIP. Hold mouse to build pressure (slow ramp). Release for spring snap-back with overshoot. AUTO mode for hands-off.
HOLD PRESSURE · M MODE · A AUTO · H HUD
06 / SWARM NEW
Swarm Field
Many marks tiling the canvas. Brand as language.
24 (adjustable) copies of the full mark across the canvas, each alive. Five layouts: SCATTER, GRID, CONCENTRIC, ORBIT, STREAM. Hold mouse to flock marks toward cursor.
M LAYOUT · HOLD FLOCK · +/− COUNT · H HUD
07 / LETTERS
Letters from Rings
The mark becomes A · R · E · N · A.
Four stadium primitives rearrange between six states: canonical mark + 5 letterforms. The ring band fills in as it morphs into a letter stroke. Same DNA, different conformations.
AUTO CYCLE · N NEXT · C CANONICAL · H HUD
08 / COMPOSER NEW
Multi-Mark Composer
Per-ring controls + place multiple marks + URL sharing.
Composer canvas hosts N marks. Each mark has independent ring sliders (R1–R4 tabs), drag to position, click to select. URL hash captures the whole composition; localStorage Library saves named configs.
N ADD · D DUP · DEL REMOVE · DRAG MOVE
09 / GENERATOR NEW
Pattern Generator
Parametric replicator: wave, trail, field, radial.
Tile the canonical mark across the canvas using four pattern modes — sine wave, linear-perspective trail, tessellated field, radial orbit. Each mode has its own slider schema. Optional opt-in motion. Freeze frame + export.
SLIDERS · SPEED · FREEZE · EXPORT
10 / ENGINE NEW
Ring Engine
One mark, many rings (1–60), combinable motion + screensaver.
A single stack with ring count as the hero. Rotate, anchor off-canvas, adjust spacing curve. 8 motion behaviors (Tilt, Breathe, Cascade, Counter, Drift, Shift, Grow, Morph) — each toggle + speed, fully combinable. Auto-Cycle screensaver mode blends through 9 curated motion states.
SLIDERS · MOTION TOGGLES · AUTO CYCLE · FREEZE
